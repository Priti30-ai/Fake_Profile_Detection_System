# 🚀 Fake Profile Detection System

## 📌 Overview

The **Fake Profile Detection System** is a full-stack application that identifies fake social media profiles using machine learning.
It analyzes user metadata and predicts whether a profile is **Real or Fake**, helping improve online safety and trust.

---

## 🎯 Key Highlights

* 🔍 Achieved **~94% accuracy** using a Random Forest model
* ⚡ Real-time prediction using integrated ML model
* 🌐 Full-stack implementation (Frontend + Backend + AI Model)
* 🧠 Supports both **Username-based** and **Manual input** prediction

---

## 🛠️ Tech Stack

**Frontend:** React.js
**Backend:** Node.js (Express)
**AI/ML Model:** Python, Scikit-learn
**Database:** MySQL

---

## ✨ Features

* ✅ Fake / Real profile classification
* 📊 Risk score generation
* 🧾 Prediction history storage (Admin dashboard)
* 🔄 REST API integration between frontend & ML model
* 🎯 User-friendly interface with dual input modes

---

## 🧠 How It Works

1. User enters username or profile details
2. Backend sends data to Python ML model
3. Model processes features and predicts result
4. Output displayed as **Fake or Real + Risk Score**
5. Data stored in database for tracking

---

## 📊 Model Details

* Algorithm: Random Forest Classifier
* Accuracy: ~94%
* Features Used:

  * Followers / Following ratio
  * Username & full name patterns
  * Account privacy & activity indicators
